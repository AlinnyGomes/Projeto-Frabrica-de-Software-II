# Empréstimo Jardim Colorado

Sistema web desenvolvido para centralizar e gerenciar as atividades da equipe de Tecnologia da Informação das instituições SESI e SEDUC.

## Sobre o Projeto

O Empréstimo Jardim Colorado foi criado com o objetivo de substituir controles manuais e planilhas utilizadas na gestão de equipamentos e demandas do setor de TI.

A plataforma reúne em um único ambiente funcionalidades para controle de empréstimos, gerenciamento de estoque, chamados técnicos, armazenamento de credenciais e acompanhamento de tarefas da equipe.

## Funcionalidades

### Empréstimos de Equipamentos

* Registro de empréstimos para alunos, professores e funcionários;
* Controle de devoluções;
* Histórico completo de movimentações;
* Identificação de equipamentos em uso.

### Chamados Técnicos

* Abertura de chamados;
* Registro de descrições e observações;
* Upload de imagens;
* Controle de status (Aberto, Em Andamento e Resolvido).

### Controle de Estoque

* Cadastro de equipamentos;
* Controle de disponibilidade;
* Equipamentos em manutenção;
* Estatísticas de utilização.

### Cofre de Senhas

* Armazenamento de credenciais;
* Registro de URLs e observações;
* Consulta rápida de acessos utilizados pela equipe.

### Tarefas

* Criação de tarefas;
* Definição de responsáveis;
* Controle de prazos;
* Monitoramento de pendências e atividades concluídas.

## Instalação

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

Acesse a pasta:

```bash
cd emprestimo-jardim-colorado
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
